# Tech Store - Project Structure

This project is organized into the following directories:

- **frontend/**: User-facing pages, assets, and includes
- **backend/**: API endpoints, admin panel, and configuration
- **database/**: Database setup and migration files
- **docs/**: Documentation and README

To run the project, access `http://localhost/tech-store2/` which will redirect to the frontend.

For detailed documentation, see [docs/README.md](docs/README.md).
